SNUG

Sorocaba Node User Group

Bem-vindo ao SNUG - Sorocaba Node User Group.

SNUG é um grupo de usuários Node.js de Sorocaba voltado para a comunidade local de desenvolvedores. Nós nos reunimos mensalmente para conversar sobre Node.js, compartilhar conhecimento e desenvolver nossa rede de contatos.

Objetivo

Colaborar com o desenvolvimento da comunidade Node.js na região de Sorocaba, ampliar a difusão do conhecimento entre os participantes, desenvolver projetos voltados para a comunidade, organizar eventos, cursos e palestras de temas relacionados à programação Javascript server-side.
